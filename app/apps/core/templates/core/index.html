🚀 快速开始

触发数据抓取：

curl -X POST http://127.0.0.1:8000/v1/parse/ \
  -H "Content-Type: application/json" \
  -d '{"tables": ["*"]}'

查询鱼获纪录：

curl http://127.0.0.1:8000/v1/records/abs/gl/records/
curl http://127.0.0.1:8000/v1/ratings/gl/
curl http://127.0.0.1:8000/v1/winners/gl/records/

带筛选条件：

curl "http://127.0.0.1:8000/v1/records/abs/gl/records/?fish= Pike&min_weight=10"
curl "http://127.0.0.1:8000/v1/ratings/gl/?in_days=true"
